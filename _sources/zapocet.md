# Formulace a reformulace modelů

Napište vždy původní model a modifikovaný model. Buďte připraveni popsat, v čem
požadovaná modifikace spočívá a jaká je interpretace přidávaného člene.

* Do modelu konkurence dvou populací přidejte třetí druh, který je konkurentem
  první populace a nijak se navzájem neovlivňuje se druhou populací.
* Do modelu konkurence dvou populací přidejte třetí druh, který je predátorem
  pro obě populace.
* Do modelu konkurence dvou populací přidejte Alleeho efekt pro obě populace.
* Do modelu konkurence dvou populací přidejte lov konstantní rychlostí pro jednu
  z populací.
* Vysvětlete předpoklady modelu logistického růstu a jak tento model rozšíříme
  na dvě konkurenční populace.
* Do modelu predátor-kořist přidejte lov konstantní rychlostí pro populaci predátora.
